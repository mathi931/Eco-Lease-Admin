# Eco-Lease-Admin

### built with
**.NET framework windows forms**

### used packages
- ADGV(advanced data grid view), 
- FastMember, FluentValidation, 
- IronPdf, 
- WebApi.Client, 
- Newtonsoft.Json

### function
- calculate and presents statistics,
- manages fleet,
- reservations, 
- customers of the lease system
